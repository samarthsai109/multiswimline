• Dynamic Table Usage Instructions
  • select the channelType by selecting the dropdown (Select an option)
  • Click on "New Step to add dynamic columns to the table"
  • Repeat the above steps to add multiple(duplicate allowed) columns/rows to the table
  • Click on the Table cell to update data into the cell
  • When clicked on the Table cell, Form will be visible based on the type of Channel clicked
  • Click on submit data to save data to LocalStorage and update the table cell as well
  • By clicking on the cell, row will also be Highlighted
  • Use UP/DOWN key arrow to view the updated data
